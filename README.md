# SafeRoute

A women's safety PWA built with React + Vite (frontend) and Node.js + Express + MongoDB (backend).

## Setup Instructions

### Frontend (Client)

1. `cd client`
2. `npm install`
3. `npm run dev`

### Backend (Server)

1. `cd server`
2. `npm install`
3. Set up `.env` file based on `.env.example` (or configure MONGODB_URI and other keys)
4. `npm run dev`
